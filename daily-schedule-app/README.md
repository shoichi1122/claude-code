# 📋 今日の予定まとめアプリ

その日の予定を登録・管理して、まとめをワンクリックでコピーできるシンプルなWebアプリです。

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバーやインストールは不要です。

```bash
# 例: そのままブラウザで開く
open daily-schedule-app/index.html        # macOS
xdg-open daily-schedule-app/index.html    # Linux
```

## 機能

- **予定の追加**: 時刻とタイトルを入力して登録(時刻順に自動ソート)
- **サマリー表示**: 予定数・完了数・残り数と「次の予定」を常に表示
- **現在の予定をハイライト**: いま進行中の予定が青枠で強調表示(1分ごとに自動更新)
- **完了チェック**: チェックボックスで完了管理
- **まとめをコピー**: 今日の予定一覧をテキスト化してクリップボードへ(SlackやLINEにそのまま貼り付け可能)

  ```
  📋 7月3日(金) の予定(1/3 完了)
  ✅ 09:00 朝会
  ⬜ 13:00 チームミーティング
  ⬜ 18:00 ジム
  ```

- **自動保存**: ブラウザの localStorage に保存。日付が変わると前日のデータは自動でクリアされ、新しい1日として始まります
- **ダークモード対応**: OSの設定に自動で追従

## 技術構成

- 単一の HTML ファイル(HTML / CSS / Vanilla JS)
- 外部ライブラリ・ネットワーク通信なし
- データはすべてブラウザ内(localStorage)に保存
