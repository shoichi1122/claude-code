# 📨 SNSリファラル管理アプリ

ダンスイベント・エステイベント・ランウェイイベントなど、イベントへの誘い(リファラル)文面をSNS別に管理するWebアプリです。

- **Instagram用**: こちらから送るDMの文面
- **Threads用**: 投稿に反応が来た人へ返信する文面

を分けて登録でき、相手の名前やイベント情報を自動で差し込んでワンタップでコピーできます。

## 使い方

公開版: **https://shoichi1122.github.io/claude-code/referral-sns-app/** — 開くだけで使えます。

ローカルで使う場合は `index.html` をブラウザで開くだけで動きます(データはブラウザのlocalStorageに保存されます)。

## 機能

### 📝 文面テンプレ
- **Instagram(こちらから送る文)/ Threads(反応が来た人に送る文)** の2チャンネルでテンプレートを管理
- イベント種別(ダンス・エステ・ランウェイ+自由に追加可)ごとに整理でき、種別チップで絞り込み
- 差し込みタグ対応: `{名前}` `{イベント名}` `{日付}` `{時間}` `{場所}` がコピー時に自動で置き換わります
- 初回起動時に3種別 × 2チャンネル分のサンプル文面入り(自由に編集・削除可)

### 🎪 イベント
- イベント名・種別・日付・時間・場所を登録
- イベントごとに「声かけした人数」「参加確定人数」を表示

### 📤 送信リスト
- 送る相手(アカウント名)を、送り方(Instagram / Threads)と誘うイベントを選んで登録
- 「💬 文面を作る」でテンプレートに相手の名前・イベント情報を差し込んだ文面をプレビュー → コピー
  - コピーすると自動でステータスが「送信済み」に変わります
- ステータス管理: 未送信 → 送信済み → 反応あり → 返信済み → 参加確定 / 見送り
- ステータスごとの絞り込みと、未送信・やり取り中・参加確定のサマリー表示

### 💾 バックアップ
- 画面下部のボタンからJSONでエクスポート/復元できます(ブラウザや端末をまたぐときに)

## データについて

すべてのデータはブラウザのlocalStorageにのみ保存され、外部には送信されません。
